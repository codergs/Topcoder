=================== BUILD ==============
run the command:
"ant package"l

then the target jar file is built, and it is located in:
build/dist/CRSNOJavaTool.jar

=================== USAGE ==============
run the command:
"java -jar build/dist/CRSNOJavaTool.jar"
then input the template file, report output file, and experimental file folder(Directory) name as the prompt message says.

Find the outputReport.xlsx under "output" folder

See README.md more more details
